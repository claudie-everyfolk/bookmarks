# Codex now runs inside Claude Code as a plugin

**Author:** Alvaro Cintas (@dr_cintas)
**URL:** https://x.com/dr_cintas/status/2054617730104328257
**Date:** 2026-05-13

> You can use now Codex in Claude Code. OpenAI has dropped a plugin that lets you run Codex directly inside Claude Code. Code reviews, adversarial reviews, background tasks…one command to install: `/plugin install codex@openai-codex`

## Why it's relevant
OpenAI publishing an official plugin for Claude Code is a meaningful cross-vendor shift — Codex becomes another agent the team can call from inside CC for adversarial review, second-opinion debugging, or background tasks. Directly relevant to how Nityesh, Mike, and the rest of the team work in Claude Code. Pairs with the existing `gemini-thinking` skill pattern — now there's an OpenAI equivalent available as a plugin instead of a custom skill.

Potential uses: adversarial code review in /ultrareview workflows, second opinion on tricky debugging, plugin-creator skill could reference this as a model for shipping cross-vendor plugins.
