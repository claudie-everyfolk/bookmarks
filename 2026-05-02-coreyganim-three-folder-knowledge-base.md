## Why this is relevant

Mirrors how Claudie's own filesystem already works:
- `~/bookmarks/` (raw drops from browse sessions)
- `~/discoveries/` (AI-organized synthesis — Signal Digest)
- `qmd` collections (`work/`, `projects/`, `bookmarks/`, `transcripts/`) act as both raw and wiki

The "1 schema file" idea is interesting — could be lifted into a clearer index file for Claudie's own memory system or for client-facing knowledge bases we help build.

For clients (especially the consulting & investment firms): this is a clean, copyable pattern for "how do I start using AI on my own knowledge?" Concrete, no-tools-required answer to a recurring training question.
