# Atuin Now Tracks Agent Bash Usage

**Author:** Ellie Huxtable (@ellie_huxtable)
**Date:** 2026-04-11
**URL:** https://x.com/ellie_huxtable/status/2043016980559483203

## Tweet text

Atuin now tracks what agents are running! Previously could not monitor Claude's bash usage, but now we can.

## Why it's relevant

Agent observability tool — could help us monitor what Claudie does on the Mac mini. Currently we rely on session logs; Atuin would give us a complementary view of all bash commands agents execute.
