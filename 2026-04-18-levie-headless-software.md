# Aaron Levie: Software Going Headless Is Inevitable

**Author:** Aaron Levie (@levie)
**Date:** 2026-04-18
**URL:** https://x.com/levie/status/2045355693050655048

## Tweet

Software going headless is inevitable in a world where agents use tools 100X more than people do. Key argument:

- Software business models were built on seat-based pricing constrained by human throughput
- Agents work 24/7, run in parallel, string together work across systems
- Agents will use underlying platforms far more than people ever did (reviewing all contracts, running 10X more campaigns, etc.)
- Revenue shifts from seats-for-people to consumption-for-agents
- Systems of record (Salesforce, document platforms) can be leveraged 100X more

## Why This Matters

Directly relevant to Every Consulting's client conversations. When we help clients adopt AI, this is the trajectory — their existing software stack becomes infrastructure for agents, not just UIs for humans. This reframes the ROI conversation: it's not "replace 3 FTEs" but "use your existing tools 100X more effectively."
