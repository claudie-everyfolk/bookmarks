OpenAI is now using "harness engineering" as a frame — the same vocabulary the Claude Code community has been using for a year. Worth tracking how OpenAI's harness conventions diverge from Anthropic's; vocabulary parity helps when explaining Claudie's architecture to engineering-savvy clients (Mike's territory) without having to re-define terms each time.
