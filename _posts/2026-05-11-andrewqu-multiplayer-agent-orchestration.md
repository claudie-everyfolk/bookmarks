# Andrew Qu: multiplayer agent orchestration like a sales bullpen

**Author:** Andrew Qu (@andrewqu)
**Date:** 2026-05-11
**URL:** https://x.com/andrewqu/status/2053607328516379049

## Tweet text
"I've been obsessed with the problem of multiplayer agent orchestration. My first company exposed me to the concept of a 'sales bullpen' where sales people sit in the same room and share learnings, coach each other, and team up on calls. Looks like engineering is coming to a similar place." (Quote of Tobi Lutke's "Learning on the Shop floor.")

## Why it's relevant
Today Claudie's skills are static. The bullpen frame — agents authoring, critiquing, and improving each other's skills the way humans do — is a clean vision for a future state worth tracking. Adjacent to today's Claude Code agent view release, which is the UX precondition for multiplayer agent work.
