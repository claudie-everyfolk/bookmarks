# Wes Roth: Claude file-based memory upgrade in preview

**Author:** Wes Roth (@WesRoth)
**URL:** https://x.com/WesRoth/status/2059122352856658274
**Date:** 2026-05-26

## Tweet

> Claude is reportedly preparing a new file-based memory upgrade that would let users choose between Memory Files and Classic memory.
>
> Memory Files are described as organized notes that Claude writes during conversations and reads later when relevant.

## Why it's relevant

Claudie already runs a file-based memory system at `~/.claude/projects/-/memory/` with `MEMORY.md` as an index — Anthropic is moving consumer Claude in the same direction. Worth watching: if the user-facing version standardizes on a particular structure or surfaces an editor UI, we may want to mirror the format so Natalia and clients can read/edit Claudie's memory directly without learning a custom layout.
