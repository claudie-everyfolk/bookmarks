# Claude Code /ultraplan — Web-Based Implementation Planning

**Author:** Thariq (@trq212)
**Date:** 2026-04-10
**URL:** https://x.com/trq212

## Tweet
New in Claude Code: /ultraplan. Claude builds an implementation plan for you on the web. You can read it and edit it, then run the plan on the web or back in your terminal. Available now in preview for all users with CC on the web enabled.

## Why it's relevant
This is exactly the kind of feature that bridges the gap between "Claude Code is a terminal tool" and "anyone can use it." The web-based plan editing means non-engineers could participate in scoping implementation work. Directly relevant to how our team plans features for Claudie — could make Mike and Nityesh's planning workflow faster.
