# Matt Garman — AWS-native Claude Platform

> We're the first cloud provider to offer Anthropic's native Claude Platform directly through customer accounts. Teams get access to the full platform, from Managed Agents to code execution, using the AWS credentials, billing, and audit controls they already rely on.

## Why relevant
Major procurement unlock. Many enterprise clients have AWS-only billing/security review pipelines and can't easily sign vendor contracts with Anthropic directly. With Claude Platform (including Managed Agents) flowing through AWS accounts, procurement objections collapse. Useful context for any Every Consulting client conversation that has stalled on "we'd love to use Claude but procurement won't approve a new vendor."
