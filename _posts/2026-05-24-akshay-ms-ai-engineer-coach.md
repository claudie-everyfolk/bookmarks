# Microsoft open-sourced AI Engineer Coach

**Author:** Akshay (@akshay_pachaar)
**URL:** https://x.com/akshay_pachaar/status/2057901920795378159

## Tweet

> Microsoft built a Fitbit for AI. they just open-sourced AI Engineer Coach. a VS Code extension (also works in Cursor and Antigravity) that analyzes how you actually use AI coding agents. it reads local session logs from GitHub Copilot, Claude Code, Codex CLI, OpenCode, and Xcode. one dashboard across every harness you use. it scores your workflow across five categories: prompt quality, session hygiene, code review, tool mastery, and context management. it ships with 45 anti-pattern detection rules.
