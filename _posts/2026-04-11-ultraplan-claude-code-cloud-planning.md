# Anthropic /ultraplan for Claude Code

**Author:** Wes Roth (@WesRoth), citing Thariq (@trq212)
**Date:** 2026-04-11
**URL:** https://x.com/WesRoth/status/2042754510880432218

## Tweet text

> Anthropic rolled out /ultraplan for Claude Code in preview, a new feature that shifts the complex planning phase of AI-assisted software development from the local command line directly to the cloud. When a developer triggers the command, Claude drafts the implementation plan for you on the web. You can read it and edit it, then run the plan on the web or back in Claude Code.

## Why it's relevant

Directly relevant to our Claude Code setup. /ultraplan moves planning to the cloud — could be useful for complex client projects where we want to review plans before execution. Mike should know about this for tech vertical training.
