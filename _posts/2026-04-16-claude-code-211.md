# Claude Code 2.1.111 Released

**Author:** @ClaudeCodeLog
**Date:** 2026-04-16
**URL:** https://x.com/ClaudeCodeLog/status/2044807351929024682

## Tweet

> Claude Code 2.1.111 has been released.
>
> 35 CLI changes, 15 system prompt changes
>
> Highlights:
> - Opus 4.7 adds xhigh between high and max for finer speed-vs-intelligence tuning via /effort
> - 'avoid creating new files' rule became a preference, so new files are easier to create

## Why this matters

The xhigh effort level is directly useful for our scheduled jobs — we can tune effort more precisely to balance speed and quality. The "avoid creating new files" change is a quality-of-life improvement for skills that generate deliverables.
