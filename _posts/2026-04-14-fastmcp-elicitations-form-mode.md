## Why This Is Relevant

Elicitations in MCP allow servers to request structured input from users mid-flow — confirmation dialogs, form fields, etc. This is relevant to:

- Our MCP setup (Google Workspace MCP, qmd) — could add confirmation steps before destructive actions
- Client training on building MCP servers — elicitations are a more polished UX than pure tool calls
- The spec link is a useful reference for Nityesh when building custom MCP servers
