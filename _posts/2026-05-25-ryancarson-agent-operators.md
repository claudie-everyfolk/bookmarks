# Ryan Carson: jobs for people who are great at using agents

**Author:** Ryan Carson (@ryancarson)
**URL:** https://x.com/ryancarson/status/2058939814599172259

> I'm becoming convinced there is going to be an explosion of jobs for people who are great at using agents.
>
> There's a clear line I'm seeing between people:
> 1. Use Claude Code / Codex / Devin / etc
> 2. Use ChatGPT / Gemini / Perplexity
>
> Distinction is simple: if your agent writes persistent code, you're in group 1 and the future is v v bright.

## Why it's relevant
Clean framing for the gap that Every Consulting's tech vertical (Mike) is selling into: chat users vs. agent operators. The "persistent code" line is a sharper definition of the divide than the usual "knowledge worker vs. agent user." Useful one-liner for tech-vertical pitches and for framing training outcomes — what we're moving clients toward is group 1, not better prompting in group 2.
