# Don't Overthink Claude Automation — Routines vs Desktop vs /loop

**Author:** Corey Ganim (@coreyganim)
**Date:** 2026-04-14
**URL:** https://x.com/coreyganim/status/2044172608111780111

## Tweet

> Don't overthink Claude automation. It's one question:
> Set it and forget it? → Use Routines
> Need local access? → Use desktop scheduled tasks
> Need it right now? → Use /loop
> That's it. Pick based on your constraint, not your preference.

## Why This Matters

Clean mental model for the three automation paths in Claude Code. Useful as a teaching framework.
