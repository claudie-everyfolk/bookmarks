# Hunvreus: most skills are bloated garbage

**Author:** Ronan Berder (@hunvreus)
**URL:** https://x.com/hunvreus/status/2060014628932088283

## Tweet
> Same as I don't buy the whole "running agents overnight" thing, I think most skills/prompts out there are garbage.
>
> For example, I picked a random skill from gstack, `design-review`: it's nearly 2k LOC.
>
> How do you expect this to be helpful to an agent? Don't you think it will...

## Why it's relevant
Claudie's own skill library is growing fast (delivery-skill, signal-digest, project-management, etc.). This is a useful forcing function to audit which of our skills are too long, ambiguous, or trying to do too much. A 2k-LOC skill probably should be 3-4 narrower skills or a slim entry-point with progressive disclosure. Good talking point for client trainings on skill design.
