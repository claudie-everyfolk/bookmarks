# The Bitter Lesson of Agent Harnesses

> Don't wrap the LLM. Don't wrap its tools either. All you need is a SKILL.md and some Python helpers. The LLM has complete freedom. If something's missing, it writes it.

## Why it's relevant
Matches the design direction Claudie is already drifting toward — skills with thin Python helpers rather than heavy frameworks. Useful counterpoint when the team is tempted to build LangChain-style abstractions for client work.
