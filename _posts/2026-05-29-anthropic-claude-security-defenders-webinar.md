# Anthropic "Claude for Defenders" — Threat Intel Numbers Worth Watching

**Author:** Daniel San (@dani_avila7)
**URL:** https://x.com/dani_avila7/status/2060423201684111713

> Attended Anthropic's Claude Security: Putting Claude to Work for Defenders webinar. Some numbers worth paying attention to:
> - 80-90% of a real cyber-espionage campaign was executed agentically by AI, at a scale no human team could match (Anthropic Threat Intelligence Report 2025)
> - Dozens of hosts compromised end-to-end in cyber-range evaluations using only off-the-shelf open source tools. No custom attack tooling (Frontier Red Team, Jan 2026)
> - 500+ high-severity zero-days found by Claude in widely-used open source projects, including bugs that survived decades of fuzzing (Frontier Red Team, Feb 2026)

## Why it's relevant
The threat landscape Every's clients (Bloomberg, NYT, finance) are walking into: agentic AI now executes the overwhelming majority of a real cyber-espionage campaign, and the same capability finds zero-days at scale. This is the "AI is dual-use, defenders must adopt it too" framing for any security-conscious client conversation — and useful context for any client asking about safe agentic deployment. Not a single actionable vulnerability, so no security-channel alert; this is landscape, not an incident.
