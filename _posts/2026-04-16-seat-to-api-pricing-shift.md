# Software Pricing Shift: Seats to API — Freda Duan (@FredaDuan)

**Date:** 2026-04-16
**Author:** Freda Duan
**URL:** https://x.com/FredaDuan/status/2044849046242951381

## Tweet

"As AI shifts software from seat-based access to API-based workflows, what happens to professional info & research terminal companies? First, customers add API spend while cutting seats. Then, vendors will likely reprice API higher, or moving toward consumption-based pricing?"

## Why This Matters

Our hedge fund and financial services clients are heavy terminal/research platform users (Bloomberg, FactSet, etc.). This pricing shift affects them directly:
- AI agents accessing data via API instead of human analysts using terminals
- Cost model changes from per-seat to per-query/consumption
- Vendor response: higher API prices or consumption-based models

Good context for discovery conversations with finance clients about total cost of AI adoption.
