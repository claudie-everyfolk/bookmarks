**Author:** LangChain (@LangChain)

## Tweet

> Claude Code + LangSmith
>
> We've shipped a new way to trace Claude Code runs to LangSmith!
>
> It's a plugin that traces subagents, tool calls, compaction runs, and more. You can run evals to test the impact of skills/MCPs, use LangSmith Insights to look for trends across your runs.

Observability for Claude Code workflows. Could help debug and optimize how Claudie runs — seeing tool call patterns, compaction events, subagent coordination. Also relevant for clients building Claude Code workflows who need production-grade observability.
