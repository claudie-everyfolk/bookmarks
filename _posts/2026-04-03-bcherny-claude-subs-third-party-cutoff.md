# Claude Subscriptions No Longer Cover Third-Party Tools

**Author:** Boris Cherny (@bcherny)
**Date:** 2026-04-03
**URL:** https://x.com/bcherny/status/2040206440556826908

## Tweet

> Starting tomorrow at 12pm PT, Claude subscriptions will no longer cover usage on third-party tools like OpenClaw.
>
> You can still use these tools with your Claude login via extra usage bundles (now available at a discount), or with a Claude API key.

## Why it's relevant

Major business model change from Anthropic. Starting April 4, Claude Max/Pro subscriptions won't cover usage on third-party tools (like OpenClaw). Users need extra usage bundles or API keys. This affects anyone building on or integrating with Claude through non-first-party tools. For Every Consulting: this is relevant for client training — need to advise clients on the new cost structure if they're using third-party Claude-powered tools. Also relevant for our own setup if any of our workflows depend on third-party Claude integrations.
