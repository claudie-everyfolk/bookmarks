# Claude Platform on AWS launches with Managed Agents

**Author:** ClaudeDevs (@ClaudeDevs)
**Date:** 2026-05-11
**URL:** https://x.com/ClaudeDevs/status/2053892878167072914

## Tweet text
"We're introducing the Claude Platform on AWS. This gives developers access to the same models and features as our native API, including Claude Managed Agents! Workloads, billing, and IAM stay inside AWS. Anthropic operates the service, and all new features ship the same day they go live on the native Claude API."

## Why it's relevant
Moves Claude into the same compliance/IAM/billing posture as native AWS services while keeping feature parity. Natalia gets a direct answer to enterprise "can we get this inside our AWS account?" — yes, day-zero parity. Also relevant to the Claude Max platform-risk memory: AWS-hosted is Anthropic's preferred enterprise channel, safer than third-party wrappers.
