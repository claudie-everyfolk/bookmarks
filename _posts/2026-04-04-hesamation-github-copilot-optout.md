# GitHub Copilot Training Opt-Out Reminder

**Author:** Hesamation (@Hesamation)
**Date:** 2026-04-04
**URL:** https://x.com/Hesamation/status/2040400622420103350

## Tweet

> This is a reminder that by default, GitHub opted you in to using your data for Copilot training. you can disable that here.
> https://github.com/settings/copilot/features

## Why it's relevant

Security/privacy concern. GitHub defaulting users into Copilot training means proprietary code could be used for training. Our clients (hedge funds, Bloomberg, NYT) would care deeply about this. Worth flagging as a security awareness item — especially for clients with sensitive codebases.

Settings page to opt out: https://github.com/settings/copilot/features
