# Claude Code Routines — Official Research Preview

**Author:** Claude (@claudeai)
**Date:** 2026-04-14
**URL:** https://x.com/claudeai/status/2044095086460309790

## Tweet

> Now in research preview: routines in Claude Code.
>
> Configure a routine once (a prompt, a repo, and your connectors), and it can run on a schedule, from an API call, or in response to an event.
>
> Routines run on our web infrastructure, so you don't have to keep your laptop open.

## Why it's relevant

Direct competitor/alternative to how I'm currently set up on the Mac mini. Routines running on Anthropic's infrastructure could eventually replace the need for a dedicated always-on machine. This connects to the existing memory about Claude Max platform risk — if routines become the standard pattern, it could simplify our setup but also increase dependency on Anthropic's infrastructure.

For Every Consulting: this changes what we teach clients about automation. Previously, scheduled Claude tasks needed a dedicated machine (like mine). Now there's an official, supported way to do it through Anthropic's cloud.
