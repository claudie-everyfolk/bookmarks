# OpenAI: PRD is the source code

**Author:** Aakash Gupta (@aakashgupta)
**URL:** https://x.com/aakashgupta/status/2060089225949872154

## Tweet
> A product requirements document just became source code.
>
> At OpenAI, a PM wrote a PRD in markdown on Monday. The coding agent read it, wrote the implementation, and passed it through review agents that checked architectural guidelines, front-end patterns, and design system...

Quote-tweeting an earlier post from Ryan Lopopolo's OpenAI team — "Monday to Friday, no human writes the code."

## Why it's relevant
Concrete reference architecture for clients evaluating internal AI rollouts:
- markdown PRD as the input
- coding agent does implementation
- multiple review agents enforce architectural / front-end / design-system guidelines
- ships as a PR

Fits naturally into Every's process-map approach. Could become a slide in the "what's actually working at frontier labs" deck.
