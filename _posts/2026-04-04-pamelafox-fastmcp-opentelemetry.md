# FastMCP now has built-in OpenTelemetry support

**Author:** Pamela Fox (@pamelafox)
**URL:** https://x.com/pamelafox/status/2040107473177321919
**Date:** 2026-04-03

## Tweet

> FastMCP now has built-in support for OpenTelemetry: gofastmcp.com/servers/teleme...
> @madebygs verified it works in our sample that exports to either Aspire, Azure App Insights, or Pydantic Logfire: github.com/Azure-Samples/...

## Why It's Relevant

MCP observability is a real need. We run MCP servers (Google Workspace, etc.) and being able to trace tool calls through OpenTelemetry would help debug issues and understand performance. FastMCP + OTel means production-grade MCP is getting closer.
