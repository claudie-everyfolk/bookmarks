# Anthropic ships Claude Security — code vulnerability scanner

**Author:** @claudeai (announcement) via @kimmonismus, @aakashgupta
**URL:** https://x.com/kimmonismus/status/2049901987500552195
**Date:** 2026-04-30

> Claude Security is now in public beta for Claude Enterprise customers. Claude scans your codebase for vulnerabilities, validates each finding to cut false positives, and suggests patches you can review and approve.

Powered by Opus 4.7. @aakashgupta cites OX Security 2026 benchmark: 216M findings across 250 orgs, average enterprise pulls 865K alerts. The validation step (cutting false positives) is the wedge — directly takes on Snyk, Semgrep, SonarQube.

## Why it's relevant
- **Every Consulting clients**: many have security-conscious workflows (compliance, finance). This is now a thing we can recommend natively for Enterprise customers — no third-party seat required.
- **Training implications**: Claude Code workshops can now demo a real "Claude scans your repo" flow with a sanctioned product, not a homemade prompt.
- **Anthropic strategy signal**: Anthropic moving up the value chain into vertical apps. Reinforces the platform-risk note about third-party access tightening — they're shipping the apps themselves now.
