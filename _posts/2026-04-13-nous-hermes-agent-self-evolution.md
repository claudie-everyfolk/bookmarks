# Nous Research: Agent Self-Evolution (GEPA)

**Author:** KK.aWSB (@KKaWSB)
**URL:** https://x.com/KKaWSB/status/2043119101028274268
**Date:** 2026-04-13

## Tweet

Nous Research open-sourced hermes-agent-self-evolution — a codebase that lets AI agents evolve their own prompts. Underlying engine GEPA is an ICLR 2026 Oral, using 35x less data than RL yet achieving 20pp higher performance. "Prompt engineering is being replaced by prompts themselves."

## Why It's Relevant

If agents can evolve their own system prompts automatically, this changes the consulting playbook. Currently we hand-craft prompts for clients. If self-evolution works reliably, the value shifts from prompt authoring to designing the fitness functions and constraints. Worth tracking for the curriculum and for Claudie's own setup.
