## Why it's relevant

Chrome's DevTools MCP server now bundles real quality-check tools that agents can use directly. This is a meaningful step: instead of just navigating pages, agents can now run Lighthouse audits, detect memory leaks, and debug accessibility — all through MCP.

Directly relevant to our agent-browser setup and any client work involving web app quality. Could be integrated into our dev-browser workflow for automated QA during client demo builds.
