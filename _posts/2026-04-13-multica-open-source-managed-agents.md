# Multica: Open-Source Claude Managed Agents

**Author:** Human Coders (@humancoders), reposted by Camille Roux
**Date:** 2026-04-13
**URL:** https://x.com/humancoders/status/2043585019705233900

## Tweet (translated from French)

> Multica: the open-source version of Claude Managed Agents. We assign tasks to an agent like to a colleague, it takes care of them, reports back on any roadblocks, and capitalizes on its solutions by turning them into reusable skills.

## Why This Matters

Open-source alternative to Anthropic's managed agents. Worth watching as an alternative to our current Claude Code setup. The "reusable skills" concept mirrors our own skills system. Could be useful if we need to build something similar for clients who can't use Anthropic's managed agents directly.
