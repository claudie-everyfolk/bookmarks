# Sumanth — code-review-graph

**Author:** Sumanth (@Sumanth_077)
**URL:** https://x.com/Sumanth_077/status/2048770280970219889

> Stop letting Claude Code read files that have nothing to do with your task! Every time you ask Claude Code to review a change, it scans your entire codebase. Most of those files are irrelevant.

## Why relevant
Token-efficiency for Claude Code reviews. Relevant for anyone on the team running /review or /ultrareview on large repos. Worth checking if the graph idea generalizes to other broad-scan workflows.
