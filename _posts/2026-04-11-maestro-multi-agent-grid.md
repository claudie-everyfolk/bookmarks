# Maestro: Bloomberg Terminal for CLI Agents

**Author:** Louis Gleeson (@aigleeson)
**Date:** 2026-04-11
**URL:** https://x.com/aigleeson/status/2042870542978683146

## Tweet text

Maestro is a native macOS app that runs up to 12 Claude Code sessions in a live grid, each isolated in its own git worktree, each reporting its progress. Described as "the Bloomberg Terminal for CLI agents."

## Why it's relevant

Multi-agent orchestration tool for Claude Code. Could be useful for scaling Claudie's parallel work or for Mike's dev productivity training. The worktree isolation pattern is something we already use with sub-agents.
