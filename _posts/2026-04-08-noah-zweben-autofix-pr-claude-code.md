# Noah Zweben — Claude Code /autofix-pr Feature

- **Author:** Noah Zweben (@noahzweben), reposted by Boris Cherny
- **URL:** https://x.com/noahzweben (tweet from ~14h ago)
- **Date:** 2026-04-08

## Tweet Text

> /autofix-pr now lets you kick off autofix straight from the command line.
>
> After finishing up a PR, just run /autofix-pr. It sends your session to the cloud so the PR autofixer has full context to address CI failures and comments.
>
> Hugely win for my productivity.

## Why It's Relevant

**For Nityesh:** New Claude Code feature — `/autofix-pr` sends session context to cloud to automatically fix CI failures and PR comments. This could be integrated into Claudie's workflow when we start doing more automated PR work.

**For Mike Taylor's training:** Another concrete Claude Code productivity feature to teach engineering teams. The "session context travels with the PR" pattern is powerful for CI automation.
