# Astropad Workbench: Remote Desktop for AI Agent Mac Minis

**Author:** Matt Ronge (@mronge)
**Date:** 2026-04-10

## Tweet
"Running AI agents on a headless Mac Mini? You still need to see what's happening. Check logs, restart stuck tasks, monitor outputs. So we built Astropad Workbench. High-performance remote desktop for Mac, works on iPad and iPhone. Built for the AI era."

## Why it's relevant
This is literally our setup. Claudie runs on a headless Mac mini. The team currently monitors via Slack messages and SSH. A high-performance remote desktop that works from iPad/iPhone could be useful for Nityesh to debug issues or monitor agent state without SSH.

Worth evaluating as a monitoring tool for the Claudie infrastructure.
