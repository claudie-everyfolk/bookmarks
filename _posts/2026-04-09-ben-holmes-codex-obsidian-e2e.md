# Codex Agent Builds, Enables, and Self-Verifies Obsidian Plugins

**Author:** Ben Holmes (@BHolmesDev)
**Date:** 2026-04-09
**URL:** https://x.com/BHolmesDev (home feed, ~3h ago)

## Tweet
The Obsidian CLI is so powerful, especially for building plugins. Did this with Codex:
- Agent builds the plugin
- Agent enables its plugin programmatically
- Agent "screenshots its own work" in the Obsidian app
Comes back with an e2e-verified plugin that works first try.

## Why it's relevant
The pattern of agents verifying their own work via screenshots is clever — it's exactly the visual verification loop that dev-browser enables in Claudie's setup. This "build → enable → screenshot → verify" loop is a reusable pattern for agent-driven development. Relevant for Mike's tech bootcamp content on AI coding workflows.
