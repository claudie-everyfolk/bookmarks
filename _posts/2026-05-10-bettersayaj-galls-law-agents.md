# Gall's law applied to agent systems

**Author:** AJ (@BetterSayAJ)
**URL:** https://x.com/BetterSayAJ/status/2053229813285151113

## Tweet
> Gall's law: "a complex system that works is invariably found to have evolved from a simple system that worked."
> Also very relevant to agent systems. Most teams are trying to jump straight to autonomous complexity before they have evals, observability, or feedback loops in place.

## Why relevant
A useful name for an argument we already make in discovery: don't start with the autonomous closed loop, start with the simplest assisted version that works, then close it. Add to the discovery-to-process-map skill resource as a quotable principle. Also relevant for Nityesh-style infra discussions — Claudie itself evolved from simple working pieces (skills) rather than jumping to full autonomy.
