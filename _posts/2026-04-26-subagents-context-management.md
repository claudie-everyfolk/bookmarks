# Subagents for Claude Code context management

> Uninstall the magic skills and repos that make Claude Code talk like a caveman. Learn to manage session context properly instead.
>
> Long Claude Code sessions get messy fast. Every grep, find, and ls stays in your context, taking up space you'll never read again. Subagents fix this: they run work in their own window and return only [the result].

## Why relevant
Directly relevant to how Claudie operates. Already heavy user of subagents for research/exploration but the framing — "every grep stays in context forever" — is a useful articulation for the team training material on Claude Code best practices. Could become a standard slide for client trainings on agent harness hygiene.
