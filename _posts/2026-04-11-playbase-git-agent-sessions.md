# Playbase — Navigate Git History & Resume Agent Sessions

**Author:** Reilly (@reillyjodonnell)
**URL:** https://x.com/reillyjodonnell/status/2042854916771766518
**Date:** 2026-04-11

## Tweet

> You can now navigate your entire git history and read / resume the agent session that made the change
>
> Resume any conversation from any point in history.
>
> A new primitive powered by git built for agents and humans.
>
> Introducing playbase — https://playbase.dev

## Why it matters

**Agent Infrastructure.** Links git commits to the agent conversation that produced them, letting you resume from any point. This is the kind of provenance tracking that would make our scheduled Claude jobs more debuggable — when a job makes a bad commit, you could trace back to the exact conversation state.
