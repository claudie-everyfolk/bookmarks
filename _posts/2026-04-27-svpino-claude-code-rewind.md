# Claude Code "Summarize from here" via /rewind

**Author:** @svpino
**URL:** https://x.com/svpino/status/2048757376376017350

## Tweet
> "Summarize from here" in Claude Code is awesome:
> 1. Hit Esc+Esc or type /rewind. Opens checkpoint menu with scrollable list of all checkpoints Claude created during the session.
> 2. Pick the checkpoint you want to summarize from.

## Why it's relevant
Add to the training-prep playbook — concrete tip for clients struggling with long-session context bloat.
