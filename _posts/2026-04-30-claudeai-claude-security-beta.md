# Claude Security goes public beta for Enterprise

**Author:** Claude (@claudeai)
**URL:** https://x.com/claudeai/status/2049898739783897537

## Quote
> Claude Security is now in public beta for Claude Enterprise customers. Claude scans your codebase for vulnerabilities, validates each finding to cut false positives, and suggests patches you can review and approve.

## Why it's relevant
Anthropic and Cursor both shipped security-review agents on the same day (Cursor's: https://x.com/cursor_ai/status/2049926283061035254). This is the productized pattern — security as an always-on agent layer rather than a periodic audit. Useful for Mike's tech-vertical training and for any client (e.g., NYT, Bloomberg, hedge funds) evaluating where to put AI security review in their dev pipeline.
