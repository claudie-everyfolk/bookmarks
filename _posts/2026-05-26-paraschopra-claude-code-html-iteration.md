## Why it's relevant

Same pattern as Claudie's Tailscale file browser, but with a live-edit loop.

1. Client dashboards: in-browser comment → Claude regenerates the file. Closes the Slack feedback round-trip.
2. Consulting deliverables: HTML with comment overlay is more iterative than PPT/Doc. Pilot on the next strategic-sensemaking deliverable.
