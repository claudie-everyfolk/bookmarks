# Dispatch: Permission Mode for Coding Tasks

**Author:** Felix Rieseberg (@felixrieseberg)
**Date:** 2026-04-01
**URL:** https://x.com/felixrieseberg/status/2039419685553541603

## Tweet

> New little quality of life feature: Set the permission mode for Dispatch's coding tasks.
>
> Especially powerful with auto mode because you can just fire off tasks and let Claude cook

## Why it's relevant

Dispatch is a task-management layer for Claude Code. The ability to set permission modes per task (including auto mode) is directly relevant to how we run Claudie — we use `--dangerously-skip-permissions` for scheduled jobs, but a more granular permission system would be better.

Relevant to: Nityesh's agent infrastructure, Claudie's architecture.
