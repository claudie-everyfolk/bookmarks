# GitHub Opted You In to Copilot Training by Default

- **Author:** Hèsam (@Hesamation)
- **URL:** https://x.com/Hesamation/status/2040400622420103350
- **Date:** 2026-04-05

## Tweet

> This is a reminder that by default, GitHub opted you in to using your data for Copilot training. You can disable that here.
>
> https://github.com/settings/copilot/features

## Why it's relevant

Privacy/security awareness for the team and clients. Any organization using GitHub should check this setting — especially financial services clients (Bloomberg, hedge funds) who have strict data governance requirements. Worth mentioning in security-focused training sessions.
