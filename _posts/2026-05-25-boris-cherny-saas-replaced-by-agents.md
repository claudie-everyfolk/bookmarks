# Boris Cherny: "most SaaS exists because humans are the integration layer"

**Author:** Dep (@0xDepressionn), quoting Boris Cherny — head of Claude Code at Anthropic
**URL:** https://x.com/0xDepressionn/status/2058946521215762626

> Boris Cherny: "most SaaS exists because humans are the integration layer."
>
> He goes further: "a lot of SaaS just goes away when agents can operate software directly."
>
> Because the only reason most SaaS tools exist is that humans need an interface to move data between systems. Agents don't need the interface.
>
> The people who are already running agents in parallel right now are not using the same software stack as everyone else. The gap between people who know how to actually run agents and people who occasionally prompt Claude is widening every month.

## Why it's relevant
The thesis behind half of what Every Consulting sells. When Bloomberg or the NYT asks "what changes when we adopt AI," the real answer isn't "you'll use ChatGPT more" — it's "the integration glue you're paying SaaS vendors for collapses." Worth quoting verbatim in discovery decks and AI-native workflow training. Boris being the source gives it weight internal slides can borrow.
