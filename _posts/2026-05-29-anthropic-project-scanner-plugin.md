**Author:** dan. (@danrodrigux)

> Anthropic quietly released an official plugin that scans your project and auto-sets up hooks, skills, MCP servers, subagents, and automations. Most people are only using Claude Code at 20% of its potential.

## Why this is relevant

High-leverage for Mike's tech-vertical bootcamps. The curriculum currently includes hand-walking devs through hooks/skills/MCP config — the slow part. If Anthropic ships an auto-setup plugin, the bootcamp can shift focus from "how do you wire this up" to "what should it do."

Also worth Nityesh running against the `every-consulting` repo as a sanity check — might surface skills/hooks we haven't built yet.
