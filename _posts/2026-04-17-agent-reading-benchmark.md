# Agent Reading Test: Benchmark for AI Web Content Reading

**Author:** Camille Roux (@CamilleRoux)
**Date:** 2026-04-17
**URL:** https://x.com/CamilleRoux/status/2045120092317290675
**Benchmark:** https://agentreadingtest.com

## Tweet

> A benchmark to test the ability of AI agents to read web content: truncation, SPA shell, soft 404, redirects… 10 scenarios, 20 points.

## Why it's relevant

Directly relevant to our agent infrastructure. We use multiple browser tools (WebFetch, agent-browser, dev-browser) to read web content. This benchmark tests exactly the failure modes we encounter: truncation, SPA rendering, soft 404s, redirects. Could use this to evaluate which of our tools handles which scenarios best.
