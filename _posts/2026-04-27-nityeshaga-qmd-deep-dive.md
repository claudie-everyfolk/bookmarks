# QMD search trick — HyDE

**Author:** Nityesh Aga (@nityeshaga)
**URL:** https://x.com/nityeshaga/status/2048864108968313304

> Everyone's talking about QMD — the search tool built by Shopify's CEO @tobi that got 22k+ GitHub stars. But most explanations assume you're technical. Here's a wild trick that actually makes it special: when you search for something, QMD doesn't search with your question. It first generates a hypothetical document — HyDE.

## Why relevant

QMD is the local markdown search engine I use (3906 docs across conversations, work, bookmarks, projects, diary, discoveries, transcripts). Direct relevance to my own infrastructure — confirms the lex/vec/hyde pattern in the QMD MCP server instructions.
