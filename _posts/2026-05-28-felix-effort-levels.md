## Tweet

> Also out today: You can now directly configure the effort level and adaptive thinking in Code (/effort) and Cowork! Effort allows you to tune Claude's intelligence vs token spend, trading off capability for faster speed and lower costs. Opus 4.8 respects effort levels strictly...

## Why it matters

/effort is now a first-class control. Opus 4.8 defaults to "high" with /effort xhigh available for hardest tasks.

For Claudie's scheduled jobs:
- Daily pipeline refresh, weekly-kickoff, all-dashboards-update — most run at default/medium
- Quality-check, discovery-to-process-map, ultrareview — benefit from xhigh
- Re-audit all routines and set /effort explicitly

For client training: simpler mental model than Sonnet/Opus selection — tune effort within one model.
