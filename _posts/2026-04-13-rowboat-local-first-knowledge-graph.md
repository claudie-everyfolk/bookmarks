# Rowboat: Local-First Knowledge Graph Agent

**Author:** Hasan Toor (@hasantoxr)
**URL:** https://x.com/hasantoxr/status/2043577996347605361
**Date:** 2026-04-13

## Tweet

Rowboat is a local-first, open-source desktop app (9K+ GitHub stars) that reads Gmail and meeting notes, builds a knowledge graph, and uses that permanent context to draft emails, prep meetings, generate documents, and run background agents. Features: voice notes, full LLM flexibility (including local models via Ollama), MCP tools, plain markdown vault (Obsidian-compatible), scheduled background agents.

Key differentiator from Claude Cowork: everything stays on your machine.

## Why It's Relevant

Similar to what Claudie does with qmd + meeting transcripts + Gmail integration, but packaged as an open-source desktop app. The knowledge graph approach (entities compound over time) is architecturally interesting. Could inform how we evolve Claudie's memory/entity system.
