# Agentic coding is machine learning

**Author:** François Chollet (@fchollet)
**URL:** https://x.com/fchollet/status/2053234697392754701

## Tweet
> Agentic coding is a form of machine learning. Generated code is best treated as a blackbox artifact whose behavior and generalization should be managed via empirical evaluation, like with any ML model.

## Why relevant
A clean reframing for client conversations: stop debating whether the code is "right" and start running evals. Pair with Gall's law (BetterSayAJ tweet today) when teaching the why-evals lesson. Worth weaving into the prompt-engineer or claude-api skill as a guiding principle.
