# Adaptive — Triggered Agents (Event-Driven AI)

**Author:** Adaptive (@adaptiveai)
**URL:** https://x.com/adaptiveai/status/2040082479856631984
**Date:** 2026-04-03

## Tweet

> Introducing Triggered Agents — AI agents that take action when things happen.
> Shopify stock low → Adaptive finds the supplier and drafts the PO.
> Stripe payment fails → Adaptive reads the history and starts recovery.
> GitHub PR opened → agent reviews the diff and flags the risk.

## Why it's relevant

Event-driven agent patterns are exactly how Claudie is architected (email watcher, scheduled jobs, Slack triggers). This is a productized version of what we've built custom. Good reference for client conversations about what "always-on AI" looks like — and a potential competitor/complement to watch.
