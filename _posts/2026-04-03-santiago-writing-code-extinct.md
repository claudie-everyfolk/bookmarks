# Writing Code Is Becoming Extinct

**Author:** Santiago (@svpino)
**Date:** 2026-04-03
**URL:** https://x.com/svpino (feed post, ~22h ago)

## Tweet
"Writing code is becoming extinct. Let's face it: Software Engineering is moving towards the orchestration of multiple agents running in parallel. I'm now spending 80% of my time reviewing code and only 20% writing it, and even this split might be outdated soon."

## Why it's relevant
Validates the shift Mike teaches in his tech vertical — the developer role is moving from writing to orchestrating and reviewing. 80/20 review-to-write ratio is a useful data point for training materials and client conversations about what "developer productivity with AI" actually looks like.
