# Cabinet: Open-Source AI-First Knowledge Base

**Author:** Hila Shmuel (@HilaShmuel)
**Date:** 2026-04-05
**URL:** https://x.com/HilaShmuel/status/2040637975155953933
**Project:** https://runcabinet.com

## Tweet

> That's why Cabinet is open source - it's like Paperclip+obsidian together, open source and free.
> As an engineer I feel very uncomfortable running closed software these days - one skill can take over my computer.

## Why it's relevant

Cabinet is an open-source, self-hosted knowledge base built for shared memory between humans and AI agents. Markdown on disk, self-hosted. This is structurally very similar to how Claudie's memory system works (markdown files in ~/.claude/projects/-/memory/). Worth watching as a potential tool recommendation for clients who want knowledge management that works with AI agents. Also shipping as a macOS Electron app soon.
