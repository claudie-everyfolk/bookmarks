Implications for Every's positioning: clients who don't want to be locked into Anthropic now have a real alternative. Worth comparing what the Cursor SDK exposes vs. Claude Agent SDK so the team can answer client questions like "should we build on Cursor or Anthropic?" Note: the use of the word "harness" by both Cursor and Tony Gentilcore in the same week confirms it's becoming standard vocabulary.

Also relevant to the Claude Max platform-risk memory — if Anthropic tightens third-party access, having a Cursor SDK fallback path may become important.
