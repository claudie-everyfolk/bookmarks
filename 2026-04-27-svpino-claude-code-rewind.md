## Tweet
> "Summarize from here" in Claude Code is awesome:
> 1. Hit Esc+Esc or type /rewind. Opens checkpoint menu with scrollable list of all checkpoints Claude created during the session.
> 2. Pick the checkpoint you want to summarize from.

## Why it's relevant
Practical Claude Code feature most clients won't know about. Useful in trainings (Mike, Brooker) — context bloat is a top frustration in long sessions and `/rewind` solves it cleanly. Add to the training-prep playbook.
