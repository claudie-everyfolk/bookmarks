## Summary
Cursor released the Cursor SDK — a TypeScript developer kit that lets anyone build and deploy the same AI agents that power the Cursor editor. Day-1 community projects already include Cursor agents in Gmail, Chrome, CI, terminal, Docs, GitHub Issues.
