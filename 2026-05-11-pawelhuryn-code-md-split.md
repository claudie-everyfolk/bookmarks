# Pawel Huryn: judgment goes in markdown, mechanics go in tools

**Author:** Paweł Huryn (@PawelHuryn)
**Date:** 2026-05-11
**URL:** https://x.com/PawelHuryn/status/2053840714799202652

## Tweet text
"Take every procedure you run more than twice and decompose it. Judgment goes in markdown. Mechanics go in a tool.

The split: code (deterministic) + md (judgement)."

Companion article: "The Compounding Agent: Portability, Determinism > Autonomy, Context > Control"
