## Tweet

> smux: a tmux config that exposes a CLI (tmux-bridge) to allow AI agents to read and write in any pane. For example, Claude Code can control Codex in the adjacent pane, and vice versa.

## Why This Matters

Multi-agent orchestration via terminal multiplexing. This is relevant to how Claudie could coordinate with other agents or tools running in parallel on the Mac mini. Could enable Claude Code instances to supervise or hand off work to each other.
